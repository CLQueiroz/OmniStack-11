# OmniStack-11
## Projeto realizado na semana OmniStack 11
 Projeto realizado reunindo uma stack podereosa com tudo que a de novo em tecnologia
### NodeJS  ReactJS  React Native

![](img/bethehero.jpg)

## Development setup
OS X & Linux:

backend 
```sh
$ cd backend
$ yarn
$ yarn start
```

frontend
```sh
$ cd frontend
$ yarn
$ yarn start
```

mobile
```sh
$ cd mobile
$ yarn
$ yarn run-android
```
